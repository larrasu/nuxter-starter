# Nuxter Starter

## Features

- Nuxt 4
- Nuxt ESLint
- NuxtHub
- Nuxt Auth
- Nuxt Content
- Nuxt UI
- Nuxt Fonts
- Nuxt Image
- VueUse

## Use as Template

```sh
npx nuxi@latest init ddt-app -t github:larrasu/nuxter-starter
```

```sh
bunx nuxi@latest init ddt-app -t github:larrasu/nuxter-starter
```
